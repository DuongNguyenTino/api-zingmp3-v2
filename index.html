Home Page

$ yarn install # npm install

Run App

$ yarn start # npm run start

Usage

Open Browser http://localhost:4000

Get FullInfo Song:

http://localhost:4000/api/v2/full-song?id={param-id-song}

Get Song:

http://localhost:4000/api/v2/song?id={param-id-song}

Get Section Bottom:

http://localhost:4000/api/v2/section-bottom?id={param-id-playlist}

Get Detail Playlist:

http://localhost:4000/api/v2/detail-playlist?id={param-id-playlist}

Get Home:

http://localhost:4000/api/v2/home?page={param-page}

Get Top 100:

http://localhost:4000/api/v2/top100

Get Charthome:

http://localhost:4000/api/v2/chart-home

Get New Release Chart:

http://localhost:4000/api/v2/new-release-chart

Get Info Song:

http://localhost:4000/api/v2/info-song?id={param-id-song}

Get Artist:

http://localhost:4000/api/v2/artist?alias={param-name}

Get Lyric:

http://localhost:4000/api/v2/lyric?id={param-id-song}

Get Search:

http://localhost:4000/api/v2/search?keyword={keyword}

Get Video:

http://localhost:4000/api/v2/video?id={param-id-video}

Get Week Chart:

http://localhost:4000/api/v2/week-chart?id={param-id-chart}

Get Hub Home:

http://localhost:4000/api/v2/hub-home

Get Hub Detail:

http://localhost:4000/api/v2/hub-detail?id={param-id-hub}

Get Radio:

http://localhost:4000/api/v2/radio

Get Section Relative Video:

http://localhost:4000/api/v2/section-relate-video?id={param-id-video}
